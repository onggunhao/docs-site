Trezor User Guide

This guide will show you how to use your Trezor Model T hardware wallet with Binance Chain and Binance DEX. Please follow best security practices when using any hardware wallet to store cryptocurrency.

Please note that the Trezor Model One is not supported yet.

Requirements

In order to use a Trezor with Binance Chain and Binance DEX, you will need:

  • A supported web browser: Chrome or Firefox
  • Access to the Binance Chain web wallet (https://binance.org/unlock) using your web browser
  • An initialized Trezor Model T device with the most recent firmware updates installed
  • The Trezor Bridge application installed on your computer

App Installation Instructions

  1. Install Trezor Bridge

Please make sure that you have bridge installed on your computer.

  1. Install the firmware

Please make sure your Trezor has the latest firmware installed.

  1. Generate and backup your address

Please make sure that you have seed phrase backed up.

Setup/Login Instructions

  1. Open Binance Chain web wallet <www.binance.org>, then go to unlock page. You should see that Trezor is already an option for you.

  1. Allow access of your devide:

  1. Export Binance address for account #1

  1. Get your address for this device

  1. Confirm your address on Trezor

Please note that Trezor only supports the Binance Chain mainnet

  1. Click on the “Balances” navigation button in the Trading Interface to view your account balances.

  1. Your account balances are displayed.

How to send Binance Chain crypto assets

  1. Click on the “Balances” navigation button in the Trading Interface to view your account balances.

  1. Then click on the “Send” button for the asset that you would like to send.

  1. Confirm to give permission

  2. Check details about this transaction

  1. You’ll see this pop-up to ask to hold the screen to sign the transaction on your Trezor.

How to trade Binance Chain crypto assets

  1. You can place your order on trading page

  1. Confirm to give permission

  1. Check details about this transaction

  1. You’ll see this pop-up to ask to hold the screen to sign the transaction on your Trezor.
